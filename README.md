### Project Overview

**Yassin Hack Tools** is a comprehensive penetration testing suite that transforms Ubuntu systems into powerful ethical hacking environments. This project demonstrates my understanding of cybersecurity principles, automation, and the integration of complex security tools into a cohesive platform.

### Important Notice

This tool is designed exclusively for **educational purposes** and **authorized security testing**. Users are fully responsible for ensuring they have proper authorization before conducting any security assessments. Unauthorized testing of systems you do not own or have explicit permission to test is illegal and unethical.

### Quick Start

```bash
# Clone the repository
git clone https://github.com/yasohm/yassin_hackTools.git
cd yassin_hackTools

# Make executable and run
chmod +x run.sh
./run.sh
```

### Core Capabilities

**Network Analysis** - Comprehensive network scanning and discovery using Nmap for infrastructure assessment.

**Web Application Security** - Advanced testing with SQLMap and RED_HAWK for identifying web vulnerabilities.

**Exploitation Framework** - Integrated Metasploit for penetration testing and security research.

**Credential Testing** - Password security assessment using Hydra and Cupp for authentication analysis.

**Intelligence Gathering** - Information reconnaissance with IPGeoLocation and D-TECT for OSINT operations.

**Security Awareness** - Social engineering simulation tools like Weeman for security training.
